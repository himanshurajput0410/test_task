# Next.js Project

A modern web application built with Next.js, React, and SCSS.

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```
## Project Structure

- `app/` - Application pages and layouts
- `components/` - Reusable React components
- `styles/` - SCSS stylesheets
- `public/` - Static assets

## Technologies

- Next.js 16
- React 19
- TypeScript
- SCSS
